Google Cloud Speech API
============================================================================

The Google Cloud Speech API dragonfly engine can be used as an
alternative to the DNS and WSR engines. It should work on Windows, macOS
and Linux through `Aenea`_.

In order to use the engine, you will need to `install the Google Cloud
SDK`_ and then `obtain application credentials`_ using:

   gcloud auth application-default login

To install *dragonfly* and the dependencies for the engine use the
following:

   python setup.py install

You may need to `install the Python headers`_ if ``Python.h`` cannot be
found.

Youâ€™ll then need to copy the *dragonfly_cloud_loader.py* script from
*dragonfly/examples* into the folder containing your grammar modules and
run it using:

   python dragonfly_cloud_loader.py

This is the equivalent to the *MacroSystem/Core* directory that NatLink
uses to load grammar modules.
